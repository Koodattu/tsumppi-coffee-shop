## Project description

An android student services coffee shop mobile application for Sein√§joki University of Applied Sciences students. Mainly written in java with user interfaces defined in xml. Project was made during my time as a student in SeAMK when there was no official student services coffee shop mobile application.

The application fetches catalogue data from a google sheets file and social media posts from Facebook and Instagram pages.

### Features
* Coffee shop products catalogue and latest social media posts
* Catalogue data is fetched from google sheets
* Latest social media posts are fetched from Facebook and Instagram
